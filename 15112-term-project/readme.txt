Name: Emoji Spritesheet Maker

Short description:
The Emoji Spritesheet Maker allows the user to make a customizable emoji sprite. 
The user chooses the face shape, hairstyle and color, eyes, eyebrows, nose, and mouths of the emoji. 
Depending on the mouths chosen, a spritesheet will be generated. 
The user also has the option to draw on the spritesheet after it is generated.

How to run the project: Run tp3.py

Libraries needed to be installed:
cmu_112_graphics (included, modified for my computer)
opencv
tkinter
module_manager from 112 website
PIL

List of shortcut commands that exist:
None
Use 'h' to get help
